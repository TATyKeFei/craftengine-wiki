# 不同版本间区别

| 版本              | 说明                                                                                     |
| :---             | :---                                                                                     |
| 社区版            | 仅正版服务器使用，供小型服务器、体验测试、创作者等使用                                       |
| Github开源版本    | 所有代码开源，但需要自行构建插件jar文件                                                     |
| 付费版            | 可加入官方DC频道、QQ售后群服务第一时间反馈解答问题、获得插件jar文件，或者吹水吹牛逼             |
